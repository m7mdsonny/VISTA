# الأمن والحماية

## التهديدات المتوقعة
- إساءة استخدام واجهات التحقق.
- محاولات الوصول غير المصرح به.
- تعديل الإشارات يدوياً.

## الإجراءات المطبقة
- Sanctum للمصادقة.
- Rate limiting على واجهات التحقق.
- رفض Webhooks غير الموقعة.
- سياسات تمنع إنشاء/تعديل الإشارات يدوياً.
- عدم تخزين أي أسرار في المستودع.
