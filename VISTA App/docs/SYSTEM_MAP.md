# خريطة النظام (Vista)

## المكونات
- **تطبيق Flutter**: العميل النهائي (موجود مسبقاً) ويستهلك واجهات API عبر JSON.
- **Backend API (Laravel)**: يخدم البيانات والاشتراكات والتنبيهات.
- **محرك التحليل**: مهام مجدولة لحساب المؤشرات وتوليد الإشارات.
- **لوحة التحكم**: إدارة الخطط والإعدادات بدون التدخل في إشارات الأسهم.

## تدفق البيانات
1. المزودات (MarketDataProvider) توفر بيانات الشموع اليومية.
2. خدمة جودة البيانات تتحقق من الاتساق وتمنع النشر عند انخفاض الجودة.
3. خدمة المؤشرات تحسب RSI و MA وغيرها.
4. محرك الإشارات يولد إشارات تلقائية.
5. خدمة الشرح ترفق أسباب وملاحظات عربية ثابتة.
6. يتم تسجيل إشعارات في notification_events وإرسالها عبر FCM عند التفعيل.

## نقاط الأمان
- Sanctum لتوثيق الجلسات والتوكن.
- Rate limiting لواجهات التحقق.
- Webhook signature stubs ترفض الطلبات غير الموقعة.
