# دليل لوحة التحكم

## صلاحيات الإدارة
- إدارة الخطط والميزات وإعدادات التجربة المجانية.
- ضبط أوزان المؤشرات والحدود والتنبيهات.
- إدارة النصوص القانونية والرسائل والبنرات.
- مراقبة جودة البيانات وسجل العمليات.

## قيود ممنوعة
- لا يمكن إنشاء أو تعديل إشارة لسهم بعينه.
- لا يمكن تعديل مستوى الثقة أو التوصية يدوياً.

## واجهات مطلوبة
- لوحة نظرة عامة وإحصاءات.
- إدارة الخطط والميزات.
- إعدادات التحليل والتنبيهات.
- سجل التدقيق.
